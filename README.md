# FRC Robot Control System

This repository contains the full control system code for a FIRST Robotics Competition (FRC) robot written in Java using WPILib. The project was developed as part of a competitive robotics team environment, focusing on real-time control, subsystem architecture, and autonomous behavior for team 11436 I was the only devolper of the code and it was my first year ever doing frc it was also our teams first year as we were a rookie team with under 15 people and 1 mentor.

## Overview

This codebase controls a functioning competition robot, including drivetrain control, subsystem management, and autonomous routines. It was designed under real constraints such as time pressure, hardware testing, and iterative debugging during competition build seasons.

## Features

- Command-based robot architecture (WPILib)
- Subsystem-based design (drivetrain, intake, etc.)
- Joystick/controller input handling
- Autonomous mode routines(very simple)
- Real-time hardware integration
- Competition-ready control structure

## Technologies

- Java
- WPILib (FRC Framework)
- Object-Oriented Programming
- Robotics Control Systems
